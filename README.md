# updated-manga
A web scraper bot built on Cheerio, Axios, and NodeMailer to check for updated manga. 

Create a constants.json file in /src before running. 

```
Sample: 

{
    "loginUser" : "",
    "loginPass" : "",
    "sendToUser" : "",
    "website" : "https://m.mangabat.com/manga-list-all", 
    "mangaTitles" : [
        "title1",
        "title2
    ]
}
```
